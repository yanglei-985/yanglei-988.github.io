<p style="margin:0 0 16px;color:#3f3d38;font-size:15.5px;line-height:1.95;text-align:justify;font-family:'Noto Sans SC','PingFang SC','Microsoft YaHei',sans-serif;word-break:break-word;">线下门店营销的本质：用户上门时<strong style="color:#2b2b28;">已经有需求</strong>（处于 C/D 点），所以比的是产品、服务和价格，最后容易陷入同质化、卷价格战。很多线下老板把这套逻辑直接搬到线上，默认刷短视频的人也有需求——这是核心误区。</p>
<p style="margin:0 0 16px;color:#3f3d38;font-size:15.5px;line-height:1.95;text-align:justify;font-family:'Noto Sans SC','PingFang SC','Microsoft YaHei',sans-serif;word-break:break-word;">真相是：<strong style="color:#2b2b28;">90% 刷短视频的人处于无需求的闲散状态</strong>（下班地铁、饭后沙发、睡前床上），只是休闲娱乐。这就是很多线下老板做短视频流量上不去、拿不到客资的根本原因。</p>

<p style="margin:0;color:#5d5952;font-size:14px;line-height:1.9;font-family:Georgia,'Noto Sans SC','PingFang SC','Microsoft YaHei',sans-serif;word-break:break-word;">误区表现：一进误区就拼命发产品介绍/干货。整个账号全是这类内容，会变成营销号——用户只看到产品、看不到创始人，信任度极低，说服成本高、客单价也做不高。产品满足需求，人物制造偏好。用户不为你这个人买单，转化就很难。</p>

<p style="margin:0 0 16px;color:#3f3d38;font-size:15.5px;line-height:1.95;text-align:justify;font-family:'Noto Sans SC','PingFang SC','Microsoft YaHei',sans-serif;word-break:break-word;">创始人讲述负债三亿、绝境创业、结合中医与 AI 开发心梗脑梗预警手表、最终还清债务并帮助三万多人的故事。用户被创业故事吸引停留 → 听故事中逐步被激发需求 → 过程中建立对创始人的信任 → 自然产生购买欲。完美完成从 A 点（无需求）到 C 点（有需求）的转化。</p>

<p style="margin:0 0 16px;color:#3f3d38;font-size:15.5px;line-height:1.95;text-align:justify;font-family:'Noto Sans SC','PingFang SC','Microsoft YaHei',sans-serif;word-break:break-word;">美国 AI 一人公司失败率高达 90%，和传统创业接近。原因是 AI 能提升产品与内容效率，却<strong style="color:#2b2b28;">解决不了获客与信任问题</strong>。商业逻辑不会因为新技术出现而改变——短视频、AI 都只是外在形式。</p>

<section style="margin:18px 0;padding:16px 18px;background:#fff;border-left:4px solid #c43b2f;border-radius:0 10px 10px 0;box-shadow:0 8px 24px rgba(0,0,0,.035);box-sizing:border-box;">
<p style="margin:0;color:#5d5952;font-size:14px;line-height:1.9;font-family:Georgia,'Noto Sans SC','PingFang SC','Microsoft YaHei',sans-serif;word-break:break-word;">讲师自身经历：一开始也没明确定位，做着做着才发现 B 端企业内训的需求，客单价能做到 10 到 30 万。若当初纠结定位，就会错过这个机会——如果我纠结定位，我在这里就死掉了。</p>

<section style="margin:42px 0 18px;padding:0;box-sizing:border-box;"><p style="margin:0 0 8px;color:#c43b2f;font-size:12px;letter-spacing:2.5px;text-transform:uppercase;font-family:'JetBrains Mono','SF Mono',Menlo,Consolas,monospace;">theme 07 · step 2 outward</p><h2 style="margin:0;color:#2b2b28;font-size:25px;line-height:1.35;font-weight:700;font-family:'Noto Serif SC','Source Han Serif SC','Songti SC',STSong,Georgia,serif;"><span style="display:inline-block;padding-left:12px;border-left:4px solid #c43b2f;">第二步 · 向外看，做差异化定价</span></h2></section>
<p style="margin:0 0 16px;color:#3f3d38;font-size:15.5px;line-height:1.95;text-align:justify;font-family:'Noto Sans SC','PingFang SC','Microsoft YaHei',sans-serif;word-break:break-word;">先看同行：随便找 20 个同行多数都能赚钱，这个赛道就能做；如果只有头部能赚钱，新手要谨慎。再看市场：优先选市场大、增速快的赛道——<strong style="color:#2b2b28;">水大才好养鱼</strong>。差异化从三维度挖：用户维度（选细分人群）、产品维度（不同功能/服务）、服务维度（额外增值服务）。</p>
<h3 style="margin:30px 0 14px;color:#2b2b28;font-size:19px;line-height:1.45;font-weight:700;font-family:'Noto Serif SC','Source Han Serif SC','Songti SC',STSong,Georgia,serif;"><span style="color:#c43b2f;font-family:'JetBrains Mono','SF Mono',Menlo,Consolas,monospace;font-size:14px;margin-right:8px;">▍</span>案例 A · 自媒体起号课程</h3>
<p style="margin:0 0 16px;color:#3f3d38;font-size:15.5px;line-height:1.95;text-align:justify;font-family:'Noto Sans SC','PingFang SC','Microsoft YaHei',sans-serif;word-break:break-word;">原价 3980 元，高不成低不就（大 IP 课 6800、普通课 999，用户看不出你强在哪）。解法不是降价，而是<strong style="color:#2b2b28;">挖差异化 + 重构产品分层</strong>：聚焦有手艺/有产品、起号就能接单的人群，拆成 999 入门版和 5980 高端陪跑版，价差 6 倍。用户心理从拒绝变成选择，转化率大幅提升。</p>
<h3 style="margin:30px 0 14px;color:#2b2b28;font-size:19px;line-height:1.45;font-weight:700;font-family:'Noto Serif SC','Source Han Serif SC','Songti SC',STSong,Georgia,serif;"><span style="color:#c43b2f;font-family:'JetBrains Mono','SF Mono',Menlo,Consolas,monospace;font-size:14px;margin-right:8px;">▍</span>案例 B · 健身私教（实体）</h3>
<p style="margin:0 0 16px;color:#3f3d38;font-size:15.5px;line-height:1.95;text-align:justify;font-family:'Noto Sans SC','PingFang SC','Microsoft YaHei',sans-serif;word-break:break-word;">原价 4999 元卡在 1500 与 6000 之间卖不动。梳理发现来的用户里<strong style="color:#2b2b28;">产后宝妈最多、痛点最痛、最容易出效果</strong>，于是聚焦产后宝妈、砍掉其他人群，增加一对一饮食定制，重构为基础版+高端版。差异化清晰后买单爽快度大幅提升。核心逻辑：找细分人群、避开同质化竞争。</p>
<section style="margin:18px 0 22px;padding:18px;background:#f7f3ed;border:1px solid rgba(120,112,100,.20);border-radius:14px;box-sizing:border-box;"><p style="margin:0 0 6px;color:#2b2b28;font-size:16px;line-height:1.6;font-weight:700;font-family:'Noto Sans SC','PingFang SC','Microsoft YaHei',sans-serif;">关键金句</p><p style="margin:0;color:#55514a;font-size:13.5px;line-height:1.8;font-family:'Noto Sans SC','PingFang SC','Microsoft YaHei',sans-serif;">创业最聪明的方式不是参与竞争，而是避开竞争。找差异化，本质就是在避开竞争。产品分三档：入门款（降门槛+筛选）、流量款（扩面+沉淀信任）、爆款（真正赚钱与交付）。</p></section>

<section style="margin:42px 0 18px;padding:0;box-sizing:border-box;"><p style="margin:0 0 8px;color:#c43b2f;font-size:12px;letter-spacing:2.5px;text-transform:uppercase;font-family:'JetBrains Mono','SF Mono',Menlo,Consolas,monospace;">theme 08 · audience</p><h2 style="margin:0;color:#2b2b28;font-size:25px;line-height:1.35;font-weight:700;font-family:'Noto Serif SC','Source Han Serif SC','Songti SC',STSong,Georgia,serif;"><span style="display:inline-block;padding-left:12px;border-left:4px solid #c43b2f;">定用户：谁掏钱，谁就是目标用户</span></h2></section>
<p style="margin:0 0 16px;color:#3f3d38;font-size:15.5px;line-height:1.95;text-align:justify;font-family:'Noto Sans SC','PingFang SC','Microsoft YaHei',sans-serif;word-break:break-word;">核心规则：<strong style="color:#2b2b28;">谁掏钱谁就是目标用户</strong>，不是你服务谁谁就是。卖小学 AI 课，掏钱的是家长 → 目标用户是家长；给被告做刑事辩护，掏钱的是家属 → 目标用户是家属。搞错方向，内容就全错。</p>

<p style="margin:0 0 8px;color:#3f3d38;font-size:15px;line-height:1.75;font-family:'Noto Sans SC','PingFang SC','Microsoft YaHei',sans-serif;"><span style="display:inline-block;width:6px;height:6px;margin:0 10px 2px 2px;background:#c43b2f;border-radius:50%;vertical-align:middle;"></span><strong style="color:#2b2b28;">问销售团队</strong>：有销售的直接问一线。</p>
<p style="margin:0;color:#3f3d38;font-size:15px;line-height:1.75;font-family:'Noto Sans SC','PingFang SC','Microsoft YaHei',sans-serif;"><span style="display:inline-block;width:6px;height:6px;margin:0 10px 2px 2px;background:#c43b2f;border-radius:50%;vertical-align:middle;"></span><strong style="color:#2b2b28;">让 AI 做用户心理分析</strong>：没时间就交给 AI。</p>

<section style="margin:42px 0 18px;padding:0;box-sizing:border-box;"><p style="margin:0 0 8px;color:#c43b2f;font-size:12px;letter-spacing:2.5px;text-transform:uppercase;font-family:'JetBrains Mono','SF Mono',Menlo,Consolas,monospace;">theme 09 · content</p><h2 style="margin:0;color:#2b2b28;font-size:25px;line-height:1.35;font-weight:700;font-family:'Noto Serif SC','Source Han Serif SC','Songti SC',STSong,Georgia,serif;"><span style="display:inline-block;padding-left:12px;border-left:4px solid #c43b2f;">定内容方向：目的 × 表达方式</span></h2></section>
<p style="margin:0 0 16px;color:#3f3d38;font-size:15.5px;line-height:1.95;text-align:justify;font-family:'Noto Sans SC','PingFang SC','Microsoft YaHei',sans-serif;word-break:break-word;">内容方向有两个维度，交叉组合成每条视频的选题。</p>

<p style="margin:0 0 8px;color:#2b2b28;font-size:15px;font-weight:700;font-family:'Noto Sans SC','PingFang SC','Microsoft YaHei',sans-serif;">维度一 · 按目的（对应 A/B/C/D）</p>
<p style="margin:0 0 6px;color:#55514a;font-size:13.8px;line-height:1.85;font-family:'Noto Sans SC','PingFang SC','Microsoft YaHei',sans-serif;"><strong style="color:#2b2b28;">A 拉新涨粉</strong> → 泛话题内容；<strong style="color:#2b2b28;">B 建立信任</strong> → 痛点型内容；</p>
