<h3 style="margin:30px 0 14px;color:#2b2b28;font-size:19px;line-height:1.45;font-weight:700;font-family:'Noto Serif SC','Source Han Serif SC','Songti SC',STSong,Georgia,serif;"><span style="color:#c43b2f;font-family:'JetBrains Mono','SF Mono',Menlo,Consolas,monospace;font-size:14px;margin-right:8px;">▍</span>两个可复用的高价值提示句</h3>

<p style="margin:0 0 8px;color:#3f3d38;font-size:15px;line-height:1.75;font-family:'Noto Sans SC','PingFang SC','Microsoft YaHei',sans-serif;"><span style="display:inline-block;width:6px;height:6px;margin:0 10px 2px 2px;background:#c43b2f;border-radius:50%;vertical-align:middle;"></span>「我说完这段话后，你先判断这句话背后的目标和结果；在执行之前，先帮我优化提示词，让最终结果更好。」</p>

<section style="margin:42px 0 20px;padding:0;box-sizing:border-box;"><p style="margin:0 0 8px;color:#c43b2f;font-size:12px;letter-spacing:2.5px;text-transform:uppercase;font-family:'JetBrains Mono','SF Mono',Menlo,Consolas,monospace;">03 · card reward</p><h2 style="margin:0;color:#2b2b28;font-size:25px;line-height:1.35;font-weight:700;font-family:'Noto Serif SC','Source Han Serif SC','Songti SC',STSong,Georgia,serif;"><span style="display:inline-block;padding-left:12px;border-left:4px solid #c43b2f;">抽卡机制：放在奖励环节，不要放在学习环节</span></h2></section>

<section style="margin:18px 0 22px;padding:18px 18px;background:#f7f3ed;border:1px solid rgba(120,112,100,.20);border-radius:14px;box-sizing:border-box;"><p style="margin:0 0 6px;color:#2b2b28;font-size:16px;line-height:1.6;font-weight:700;font-family:'Noto Sans SC','PingFang SC','Microsoft YaHei',sans-serif;">机制拆分</p><p style="margin:0;color:#55514a;font-size:13.5px;line-height:1.8;font-family:'Noto Sans SC','PingFang SC','Microsoft YaHei',sans-serif;">学习环节负责确定性：掌握、训练、反馈、纠错。奖励环节负责不确定性：积分、盲盒、稀有卡、兑换、炫耀和收藏。</p></section>

<p style="margin:0 0 16px;color:#3f3d38;font-size:15.5px;line-height:1.95;font-family:'Noto Sans SC','PingFang SC','Microsoft YaHei',sans-serif;">Silence 反复提醒：很多人学课程时第一遍会很兴奋，以为发现新大陆，但真正执行时会进入自我怀疑。原因是从「听懂」到「学会」到「习得」，中间要跨过好几座山。</p>
<section style="margin:18px 0;padding:16px 18px;background:#fff;border-left:4px solid #c43b2f;border-radius:0 10px 10px 0;box-shadow:0 8px 24px rgba(0,0,0,.035);box-sizing:border-box;"><p style="margin:0;color:#5d5952;font-size:14px;line-height:1.85;font-style:italic;font-family:Georgia,'Noto Sans SC','PingFang SC','Microsoft YaHei',sans-serif;word-break:break-word;">纸上得来终觉浅，绝知此事要躬行。不是躬行一次就行，而是要碰壁、复盘、再跑小闭环。</p></section>

<section style="margin:42px 0 20px;padding:0;box-sizing:border-box;"><p style="margin:0 0 8px;color:#c43b2f;font-size:12px;letter-spacing:2.5px;text-transform:uppercase;font-family:'JetBrains Mono','SF Mono',Menlo,Consolas,monospace;">05 · traffic logic</p><h2 style="margin:0;color:#2b2b28;font-size:25px;line-height:1.35;font-weight:700;font-family:'Noto Serif SC','Source Han Serif SC','Songti SC',STSong,Georgia,serif;"><span style="display:inline-block;padding-left:12px;border-left:4px solid #c43b2f;">流量和商业：定赛道、定产品、定用户、定内容</span></h2></section>

<p style="margin:0 0 8px;color:#3f3d38;font-size:15px;line-height:1.75;font-family:'Noto Sans SC','PingFang SC','Microsoft YaHei',sans-serif;"><span style="display:inline-block;width:6px;height:6px;margin:0 10px 2px 2px;background:#c43b2f;border-radius:50%;vertical-align:middle;"></span><strong>定赛道：</strong>从擅长、喜欢、值得三个圆的交集里找；如果没有交集，优先做擅长的。</p>
<p style="margin:0 0 8px;color:#3f3d38;font-size:15px;line-height:1.75;font-family:'Noto Sans SC','PingFang SC','Microsoft YaHei',sans-serif;"><span style="display:inline-block;width:6px;height:6px;margin:0 10px 2px 2px;background:#c43b2f;border-radius:50%;vertical-align:middle;"></span><strong>定产品：</strong>英语项目可以先理解为「用 AI 颠覆传统英语教培」的产品雏形。</p>

<p style="margin:0 0 8px;color:#3f3d38;font-size:15px;line-height:1.75;font-family:'Noto Sans SC','PingFang SC','Microsoft YaHei',sans-serif;"><span style="display:inline-block;width:6px;height:6px;margin:0 10px 2px 2px;background:#c43b2f;border-radius:50%;vertical-align:middle;"></span><strong>定内容：</strong>内容服务于目标用户，内容越垂直，人设越清晰。</p>
